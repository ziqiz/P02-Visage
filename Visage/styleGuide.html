Default Button Style

CSS selector:

.buttonLeft or .buttonRight

Sample code:

<a href="#"><div class = "buttonLeft">Default Button Left</div></a>
<a href="#"><div class = "buttonRight">Default Button Right</div></a>

Rendered element:

Default Button Left
Default Button Right

Default Navigation Style

CSS selector:

nav

Sample code:

<nav><ul><li><a href = "#">Home</a></li></ul></nav>

Rendered element:

  • Home

Default Side Navigation Style

CSS selector:

#sideNav

Sample code:

<div id = "sideNav"><h4>Title</h4><ul><li><a href = "#">item</a></li></ul></div>

Rendered element:

title

  • Item

Carracells

#direction

Sample code:

<div id = "direction"><a href = "#">Home</a>><a href = "#">Page</a></div>

Home > Page
